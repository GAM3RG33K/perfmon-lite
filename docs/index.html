Instant Start

Boot-to-profiling in under a second. No heavy IDEs, no waiting.

📱

Android & iOS

One interface for both platforms. Auto-detect devices and simulators.

📊

Live Charts

Real-time CPU, memory, and thread sparkline charts in your terminal.

📦

Export Anywhere

Export telemetry to JSON, Markdown, or HTML — perfect for CI/CD.

🔋

Zero Bloat

Single 5.5MB static binary. Zero runtime dependencies, no web calls.

🎯

Target by App

Profile specific apps by package name or bundle ID with --id.

Install in seconds

# macOS / Linux
$ curl -sfL https://get.perfmon.qzz.io | bash

Windows: iwr https://get.perfmon.qzz.io/windows -useb | iex
