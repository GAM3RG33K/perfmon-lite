Instant Start

Boot-to-profiling in under a second. No heavy IDEs, no waiting.

📱

Android & iOS

One interface for both platforms. Auto-detect devices and simulators.

📊

Live Charts

Real-time CPU, memory, and thread sparkline charts in your terminal.

📦

Export Anywhere

Export telemetry to JSON, Markdown, or HTML — perfect for CI/CD.

🔋

Zero Bloat

Single 5.5MB static binary. No runtime dependencies, no web calls.

🎯

Target by App

Profile specific apps by package name or bundle ID with --id flag.

CommandWhat it does

perfmon devicesList connected devices
perfmon --export htmlExport telemetry report
perfmon updateSelf-update to latest version
